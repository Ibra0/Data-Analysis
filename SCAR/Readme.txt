scar.py divided into 5 main components: 

1- entries_extractor: function based on PySimpleGUI class that showing the gui-widget. 
2- Tools: Tools class is containing all the importat functions for data filters and anomalies, warning and failure positions. 
3- Rauto: the main class which containing the calculations and the report generation under generate function. 
4- load: function that containing the loading page 
5- done: function that indicating the end of the generate function.
